# 安装

依赖于`MaxPHP`，所以需要先安装`MaxPHP`
```shell
composer create-project max/max . --prefer-dist
```

如果你安装了`MaxPHP`却移除了`max/framework`，需要用下面的命令安装
```shell
 composer require max/framework
```
